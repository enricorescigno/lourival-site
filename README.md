# Site Dr. Lourival Carvalho

Site institucional do Dr. Lourival Carvalho — Cirurgião Plástico Especialista.

## Tecnologias

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Como rodar localmente

```sh
npm install
npm run dev
```

O site estará disponível em `http://localhost:8080`.

## Build de produção

```sh
npm run build
npm run preview
```
